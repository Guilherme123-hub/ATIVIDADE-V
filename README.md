# ATIVIDADE PRÁTICA V - Desenvolvimento Web I

Este repositório contém a entrega da Atividade Prática V para a disciplina de Desenvolvimento Web I da Alfa UNIPAC.

**Autor:** Guilherme Martins

## Objetivo
O objetivo deste projeto é o desenvolvimento estático de uma aplicação web simulando uma rede social, utilizando o framework Pico CSS.

## Estrutura de Arquivos
- `index.html`: Página de Login
- `cadastro.html`: Página de Cadastro
- `feed.html`: Feed com postagens e cabeçalho
- `iimg/`: Diretório contendo as imagens (`avatar.jpg`, `imagem-cabecalho.jpg`)
- `styles/styles.css`: Estilos customizados adicionais

## Como executar
Basta clonar o repositório em sua máquina e abrir o arquivo `index.html` em qualquer navegador web. A navegação pelas telas ocorre a partir da própria interface gráfica (clicando em "Entrar", "Cadastrar", ou "Sair").
